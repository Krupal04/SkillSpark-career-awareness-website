# SkillSpark Career Awareness Website

## Overview
SkillSpark is a **Career Awareness Website** designed to help students explore various career options, understand career paths, and gain knowledge about personal and professional growth. The website is interactive, user-friendly, and includes login/logout functionality for personalized experiences.

---

## Features
- **Home Page:** Introduction and overview of the platform.
- **Career Exploration:** Explore different career options with guidance and resources.
- **User Login & Dashboard:** Users can log in to access personalized content.
- **Logout Functionality:** Ensures secure logout and redirects to the home page.
- **Responsive Design:** Compatible with desktops, tablets, and mobile devices.
- **Interactive Elements:** Engaging UI to enhance user experience.

---

## Technologies Used
- **HTML5** – Structure of the website
- **CSS3** – Styling and layout
- **JavaScript** – Interactive features, login/logout functionality
- **Local Storage / Session Storage** – To handle login session

## Usage
1. Open `index.html` in a browser to view the home page.
2. Navigate to login if required.
3. Explore career options and other resources.
4. Use the logout button to end the session and return to the home page.
